# Credit Card Fraud Detection

## Overview
This project detects fraudulent credit card transactions using machine learning.

Fraud detection systems help financial institutions prevent financial losses.

## Problem Type
Binary Classification

## Challenges
Dataset is highly imbalanced because fraud cases are rare.

## Model
Logistic Regression

## Evaluation Metrics
Accuracy
Confusion Matrix
Precision
Recall
F1 Score
